## Firmware Libraries

| Library | Version | Purpose |
|---|---|---|
| MAVLink | 2.0.4 | Communication protocol used for telemetry and control messages. |
| PWMServo | 2.1.0 | Controls PWM based servos from the Teensy. |
| MadgwickAHRS | 1.2.0 | Sensor fusion algorithm used for orientation estimation. |
| FastIMU | 1.2.6 | Driver library for IMU sensors such as MPU6050. |

## Runtime Software

| Software | Purpose |
|---|---|
| OpenHD | Handles wireless video transmission and control communication. |
| QOpenHD | User interface for monitoring the OpenHD system. |

## Firmware

| Component | Description |
|---|---|
| SWIGX_RCFC_MPU6050_Teensy32.ino | Main firmware running on the Teensy flight controller. |
